# 源流绿电直连自测工具

**Yuanliu Green Power Direct Connection Self-Assessment Tool**

面向新能源项目开发者的**绿电直连项目可行性自测工具**。五步引导式评估流程，基于现场踏勘数据估算发电能力、匹配用户负荷、校验政策合规性、测算碳减排效益，最终生成可打印的结构化评估报告。

在线使用：https://sourceflower.com/yuanliu-green-power （部署后更新）

仓库地址：https://github.com/SourceFlowDigital/yuanliu-green-power

---

## 项目定位

随着中国绿电直连政策持续推进（发改能源〔2026〕688号等），越来越多的新能源项目开发者需要快速评估直连项目的可行性。但目前市场上缺乏**开源、免费、可离线使用**的自测工具——现有方案要么是咨询公司高价定制，要么是闭源商业软件。

本工具由 **宁夏源流数字** 开源发布，面向中小型新能源开发商、EPC 方、用电企业能源管理人员，提供**零门槛、全浏览器端**的绿电直连自测能力。

## 功能概览

| 步骤 | 内容 | 说明 |
|------|------|------|
| **第一步 · 基本信息** | 项目名称、地点、类型（并网/离网）、能源形式（光伏/风电/生物质/组合） | 建立评估档案 |
| **第二步 · 现场踏勘** | 屋顶光伏、地面光伏、风电、生物质等多类场址信息录入，含权属确认、电力设施现状、用电客户负荷曲线（12时段×2小时） | 核心数据采集 |
| **第三步 · 资源与负荷** | 自动汇总发电能力，支持灵活性系数选择、储能配置优化预览，展示发电-负荷分时匹配图 | 源荷匹配 |
| **第四步 · 合规测算** | 基于发改能源〔2026〕688号第2条三大指标自动校验——自发自用率≥60%、自用占负荷比≥35%/30%、上网率≤20% | 一键合规判定 |
| **第五步 · 碳减排** | 自动计算年减排量（tCO₂）、全生命周期减排量（万tCO₂）、等效植树量（万棵/年） | 绿色价值量化 |

### 特色功能

- **储能优化预览**：输入储能功率/容量/年循环次数，自动推算对自发自用率和合规指标的改善效果
- **分时匹配图表**：基于 Chart.js 的 12 时段发电-负荷柱状图对比，直观展示供需匹配
- **可打印评估报告**：一键生成包含全部数据、图表、合规结论的结构化报告，支持打印/导出 PDF
- **离线可用**：纯前端单文件 SPA，无后端依赖，下载即可在浏览器运行
- **移动端适配**：暗色主题 + 移动优先设计，踏勘现场用手机即可录入数据

## 合规依据

本工具的核心合规算法基于 **发改能源〔2026〕688号** 第二条：

| 指标 | 公式 | 阈值 | 说明 |
|------|------|------|------|
| 自发自用率 | 自用电量 ÷ 总发电量 | **≥ 60%** | 必须达标 |
| 自用占负荷比 | 自用电量 ÷ 用户总负荷 | ≥ 35%（2030年前）/ ≥ 30%（2030年后） | 必须达标 |
| 上网电量比例 | 上网电量 ÷ 总发电量 | **≤ 20%** | 必须达标 |

任一指标不达标即判定为不合规，工具会给出缺口分析与补救建议。

## 技术架构

```
单文件 SPA
├── HTML/CSS（暗色主题、移动端适配）
├── Vanilla JavaScript（无框架依赖）
├── Chart.js 3.9.1（CDN，仅图表模块）
└── 零后端 / 零数据库 / 纯浏览器运行
```

- **无需安装**：下载 `index.html` 用浏览器打开即可
- **无需网络**（除 Chart.js CDN）：所有计算在本地完成，数据不上传
- **隐私安全**：用户输入的踏勘数据不离开浏览器
